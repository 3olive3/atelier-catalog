# Style Presets — Visual Diagram

Each style maps to a Nano Banana 2 prompt modifier that transforms the SVG blueprint into a specific visual aesthetic. Edit these to customize the output look.

## handdrawn (Default)

**Best for:** Presentations, YouTube explainers, blog posts

> Transform this SVG diagram layout into a beautiful hand-drawn illustration. Use a sketchy, organic line style with slight imperfections that look naturally drawn. Warm paper-like texture background. Colors should be soft and inviting. Text must be perfectly legible — use a clean handwriting-style font. The overall feel should be like a talented designer's whiteboard sketch that's been polished to presentation quality.

**Tips:** Works best with simpler diagrams (5-10 main elements). Complex diagrams with many small elements may lose detail in the hand-drawn style.

## polished

**Best for:** Professional documentation, blog hero images, reports

> Transform this SVG diagram layout into a clean, modern digital illustration. Crisp edges, flat design aesthetic, professional infographic quality. Use subtle shadows and gradients for depth. Colors should be vibrant but harmonious. Text must be perfectly legible with a modern sans-serif look. The result should look like a professional infographic from a top design agency.

**Tips:** The most reliable style for text-heavy diagrams. Handles complexity well.

## blueprint

**Best for:** Technical deep-dives, engineering docs, architecture reviews

> Transform this SVG diagram layout into a technical blueprint-style illustration. Dark navy blue background with white and light blue line art. Engineering diagram aesthetic with grid lines visible in the background. Text in a monospace or technical font style. Precise, technical feel. Like a high-end architectural or engineering blueprint.

**Tips:** Very distinctive look. Works well for system architecture and infrastructure diagrams.

## watercolor

**Best for:** Creative content, artistic features, soft presentations

> Transform this SVG diagram layout into a soft watercolor illustration. Organic shapes with color bleeding at the edges. Gentle, artistic feel. Light watercolor paper texture visible. Colors flow naturally between elements. Text should remain crisp and legible against the soft background.

**Tips:** Text legibility can be lower with this style. Use larger font sizes and fewer text elements.

## minimal

**Best for:** Elegant slides, executive presentations, posters

> Transform this SVG diagram layout into ultra-minimal line art. Black lines on pure white background. Single consistent line weight throughout. Maximum whitespace. No fills, no gradients, no shadows — just clean outlines. Text in a thin, elegant sans-serif style.

**Tips:** The simplest visual output. Best for diagrams where the structure itself is the message.
