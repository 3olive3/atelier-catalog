# Git Repository Catalog

Repository templates and references for common project structures.

Entries here define git repository configurations that can be pre-loaded into an agent's authorized repository scope.

## Entry Format

```json
{
  "catalogID": "monorepo-template",
  "name": "Monorepo Template",
  "description": "Standard monorepo layout with packages/, apps/, and shared config",
  "version": "1.0.0",
  "source": "marketplace",
  "defaultBranch": "main",
  "remoteURL": "https://github.com/example/monorepo-template"
}
```
